About the project

This project is about create and managing websocket services. To do this I created a PHP websocket server to handle the connections and manage the services. The services are launched and can get shut down without in live (without rebooting the server). Services are basically PHP classes that recieves data from clients and can send data to them or to the server also. I try to keep the clode as clean as possible and I use phpDoc and jsDoc to generate the technicals documentation. I use sonar to monitor my code and linters in my IDE and try to fix the maximum errors reported I can fix.
